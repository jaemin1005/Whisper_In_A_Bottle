## 커밋 컨벤션 
Feat - 문서변경
Fix - 수정
Docs - 문서(리드미)
Style - 코드구조 변경(주석)
Refactor - 리팩토링
Desige - 마크업 + css
New - 새파일 생성
Del - 파일 삭제
Rename - 이름 및 경로 변경
Test - jest파일 생성
Npm - 모듈관련(Npm | New , Npm | Del)
.gitignore - gitignore 수정
